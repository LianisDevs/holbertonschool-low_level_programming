# DOUBLY LINKED LISTS

* 0- prints all elements in a doubly linked list

* 1- returns the number of elements in a doubly linked list

* 2- adds a new node to the beginning of the doubly linked list

* 3- adds a new node to the end of the doubly linked list

* 4- frees a doubly linked list

* 5- returns the nth node of a doubly linked list

* 6- returns the sum of all the data (n) of a doubly linked list

* 7- inserts a new node at a given position

* 8- deletes the node at index position of doubly linked list
